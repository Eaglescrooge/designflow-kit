## Overview

UX Automation tools leverage artificial intelligence to:
- Generate wireframes and UI designs
- Convert designs to code
- Assist with user research
- Create documentation automatically
- Write UX copy

### Accessing UX Automation

1. Click **"UX Automation"** in the main navigation
2. Browse tools organized by category
3. Click any tool to open it in a new tab

### GPT Badge

Tools marked with a **GPT** badge are powered by ChatGPT and require:
- ChatGPT Plus subscription ($20/month)
- Access to GPT-4 or custom GPTs

## Usage Tips

### Getting the Best Results

1. **Be Specific**: Provide detailed prompts with context
2. **Iterate**: Use follow-up prompts to refine outputs
3. **Combine Tools**: Use multiple tools in your workflow
4. **Verify Outputs**: Always review AI-generated content

### Prompt Writing Tips

| Do | Don't |
|----|-------|
| Include context and constraints | Use vague descriptions |
| Specify target audience | Assume AI knows your brand |
| Mention design system/style | Accept first output as final |
| Request multiple variations | Skip the review process |

### Example Workflow

1. **Research**: Use QoQo.ai to generate personas
2. **Wireframe**: Use Relume to create site structure
3. **UI Design**: Use Galileo or v0 to generate interfaces
4. **Testing**: Use Attention Insight to validate designs
5. **Code**: Use Locofy or Anima to generate code
6. **Copy**: Use Writer for UX copy review

### Cost Considerations

| Tool Type | Typical Pricing |
|-----------|-----------------|
| Custom GPTs | ChatGPT Plus ($20/mo) |
| UI Generators | Free tier + paid plans |
| Design-to-Code | Free trial + enterprise |
| Research Tools | Usage-based pricing |
