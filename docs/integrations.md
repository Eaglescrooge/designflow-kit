## Using Integrations

### In DesignFlow Kit

1. Navigate to the **Integrations** section
2. Click on any integration card
3. View detailed documentation including:
   - Video tutorials
   - Setup guides
   - API documentation
   - External links

### API Authentication

Most integrations use one of these auth methods:

| Method | Description |
|--------|-------------|
| API Key | Simple key-based authentication |
| OAuth 2.0 | Token-based with user consent |
| Bearer Token | JWT or personal access token |

### Common Endpoints

Each integration detail page includes:
- Base URL
- Authentication method
- Available endpoints
- Code examples
