================================================================================
                    DESIGNFLOW KIT - DESIGN SPECIFICATIONS
                         For Figma Recreation
================================================================================

OVERVIEW
--------
DesignFlow Kit is an open-source UX/Product Design toolkit with a modern,
clean aesthetic inspired by Linear, Notion, and Figma. The design prioritizes
functionality while exemplifying excellent design standards.

================================================================================
                          PAGE STRUCTURE
================================================================================

LANDING PAGE SECTIONS
---------------------
1. Hero Section          - Full-height, two-column layout
2. Problem/Solution      - Two-column explaining the need
3. Pre-Kit Tools         - 3-column grid (10 tools)
4. Post-Kit Tools        - 3-column grid (7 tools)
5. Integration Ecosystem - Logo grid with 41+ tools
6. Community Stats       - 4-column metrics display
7. Getting Started       - Code snippet + quick start
8. Open Source CTA       - GitHub contribution call-to-action
9. Footer                - Multi-column navigation


DASHBOARD
---------
Layout:                 Single page with phase tabs
Phase Tabs:             Pre-Design Kit, Post-Design Kit
Progress Display:       Progress bar + completion count
Tool Grid:              2-column on desktop
Tool Card Actions:      Click to toggle completion


DOCUMENTATION
-------------
Layout:                 Sidebar (desktop) + Content area
Sidebar:                280px width, sticky
Tabs:                   Mobile navigation alternative
Content Width:          Max 65ch for readability
